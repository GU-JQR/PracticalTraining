Brief introduction

研究院简介

程式防治（武汉）癌症医学研究院，院长罗鹏飞教授。下辖有一个肿瘤技术中心~上海竑鹏肿瘤新技术中心，一个临床科研拓展基地~广东药科大学附属第三医院肿瘤科五区；拥有多个自有知识产权的新型发明专利，及附属产品。其中，着力于自有知识产权的国际专利“罗鹏飞程式肿瘤诊疗法”的临床病案拓展研究的“罗鹏飞•王健程式防治肿瘤科研团队”，是中国国内知名的肿瘤程式微创治疗专家团队。其临床医疗实力，不仅在业界有着很高的地位和声望，而且经患者口口相传，更是声名远播。该团队每年要为各种肿瘤患者做1500台次程式微创介入手术，积累了数万例的临床成功案例。
